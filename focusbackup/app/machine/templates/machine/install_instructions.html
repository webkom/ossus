{% block title %}Install new server{% endblock %}

1. Install Java 7

You need java installed, download java 7 before installing backup. If the downloaded version is not supported on your system, find the correct on the java website.

2. Download backup installer

Download Focus24 Installer and click to open the file.

3. Start the installer

When you have opened the installer, you will se the following picture

4. Configure

Machine Name of the machine (e.g NEDAP-HIVE)
Local admin user (e.g Administrator)
Local admin password the password for the admin user
Template ID Provide the ID for the template you want this machine to be based on.
You can select from this list of templates. Click on the desired template, find the ID and use this id in the installer.
Server Default server http://focus24.no
Focus24 user Your username for login at Server (focus24.no)
Focus24 password Your password

Example

5. Complete the installation

Click on install. Install is OK if the box disappear. If not, double check that all look OK.

6. Further configuration

Go to list of machines to find your new machine. Click on it to see if its reporting and do more configuration under "schedules".
